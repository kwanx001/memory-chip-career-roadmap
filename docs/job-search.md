# 求职执行指南

## 岗位关键词

搜索时组合使用以下词语：

- 半导体设备技术员、设备助理工程师、设备维护。
- 晶圆制造技术员、生产技术员、工艺技术员。
- 封装测试技术员、ATE 测试助理、产品测试。
- 可靠性测试、实验室技术员、失效分析助理。
- MLCC 设备、制造、品质、可靠性和检测。

## 岗位评分

每看到一个岗位，按 10 分制记录：

| 项目 | 分值 |
| --- | ---: |
| 专科学历可申请 | 2 |
| 提供设备、测试或分析实操 | 2 |
| 能接触 SPC、良率或故障数据 | 2 |
| 有明确培训或晋升路径 | 2 |
| 工作内容可形成量化成果 | 2 |

优先申请 7 分以上岗位。纯搬运、长期无设备和数据接触、没有技能积累的岗位只作为短期
过渡。

## 简历表达

不要写：

> 熟悉 Python，了解半导体制造。

改写为：

> 使用 Python 分析 5,000 个模拟 die 的测试数据，生成 wafer map、bin 分布和 CPK 报告，
> 并识别边缘失效与局部异常簇。

不要写：

> 学习过失效分析。

改写为：

> 按 8D 方法完成 NAND 编程失败与 MLCC 绝缘电阻下降案例，建立现象、遏制、根因验证和
> 预防措施闭环。

## 面试准备

必须能够回答：

1. MOS 管如何作为开关工作？
2. DRAM 为什么需要刷新？NAND 为什么不需要？
3. 什么是良率、bin、均值、标准差和 CPK？
4. 设备报警时如何保证安全并逐步排查？
5. 相关性为什么不等于根因？如何设计验证实验？
6. 讲述一个自己发现错误并修正的项目实例。

## 工作后的证据记录

每周记录：

- 遇到的设备、测试或质量异常。
- 使用过的仪器、软件和分析方法。
- 自己采取的动作及验证结果。
- 节省时间、降低缺陷或减少停机的数字。

不得带走公司机密数据。对外作品使用脱敏、重新构造或模拟数据。
